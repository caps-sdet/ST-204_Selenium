# ST-204_Selenium

Bu repository CAPS Academy-də keçilən Java dərslərinin praktiki materiallarını ehtiva edir. Burada yalnız source kodlar (src/) yerləşir. IntelliJ layihə faylları və build faylları yoxdur.

Başlanğıc Addımları

Repo-nu yükləmək git clone https://github.com/caps-sdet/ST-204_Selenium.git

Yükləməli olduğunuz kitabxanalar : 
1. org.seleniumhq.selenium:selenium-java:4.43.0
2. junit:junit:4.13.2

Dərsləri açmaq src/ qovluğuna daxil olun:

src/ ├─ Gun_01/ ├─ Gun_02/ └─ ...

Hər qovluq həmin günün dərs materiallarını ehtiva edir. Fayl adları dərsdə keçilən mövzuları göstərir:

_00_Println.java _01_Variables.java

IntelliJ-də işləmək src/Gun_XXX qovluğunu “Open Project” ilə açın. IDE spesifik fayllar yoxdur, tələbələr öz konfiqurasiyalarını istifadə edə bilərlər.

Yeni dərsləri əldə etmək Hər yeni dərs əlavə olunduqda fork-larınızı update edin.
